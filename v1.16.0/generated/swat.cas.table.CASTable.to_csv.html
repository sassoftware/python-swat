swat.cas.table.CASTable.to_csv

CASTable.to_csv(*args, **kwargs)

Write table data to comma-separated values (CSV)

This method writes a file on the client side. This means that all of the data in the table must all be fetched. If you want to save a file on the server side, use the table.save CAS action.

Parameters:
*argspositional arguments

Positional arguments to pandas.DataFrame.to_csv()

**kwargskeyword arguments

Keyword arguments to pandas.DataFrame.to_csv()
