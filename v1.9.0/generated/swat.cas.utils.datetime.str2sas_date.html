swat.cas.utils.datetime.str2sas_date¶

swat.cas.utils.datetime.str2sas_date(dts)¶

Convert a string to a SAS date

Parameters
dtsstring

The string representation of a date.

Returns
float

SAS date

See also

pandas.to_datetime()

Examples

>>> str2sas_date('19700101T12:00')
3653.0
